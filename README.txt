Proyecto LiterAlura - Catálogo de Libros

Este proyecto es parte del desafío de Alura Latam (Oracle Next Education).
Permite consultar libros desde la API Gutendex y almacenarlos en una base de datos PostgreSQL.

Funcionalidades:
1. Buscar libro por título (consulta a la API y guarda en la base de datos)
2. Listar todos los libros registrados
3. Listar todos los autores
4. Listar autores vivos en un año determinado
5. Listar libros por idioma

Requisitos:
- Java 17
- Maven
- PostgreSQL
- Spring Boot 3.2.x

Configuración:
- Editar application.properties con tus credenciales de PostgreSQL.
- Ejecutar con: mvn spring-boot:run
